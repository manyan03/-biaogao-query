# 汉字音韵地位查询工具

基于《汉字古音表稿》（上古音）与《广韵》（中古音）的汉字音韵地位批量查询工具。

## 功能特点

- 支持批量查询，一次可输入多个汉字
- 简体自动转繁体查询
- 简繁体音韵地位不同时（如"并"与"並"）同时显示两种结果
- 反切组紧邻显示：广韵反切、表稿反切、韵部拟音、表稿拼音，便于对照（如「孩」表稿"户來" vs 广韵"戶來"）
- 显示上古音（声、韵、调、开合等第）与中古音（声、韵、调、等、呼、摄）
- 集韵/又音异读分多行独立显示（如「幼」分显伊謬/yòu 与（集）一笑-笑（幼妙）/yào 两行，标注"幼妙"义的集韵异读）
- 上古声母以「字母+IPA」形式显示（如「餘ʎ」），便于识读
- 支持导出查询结果为 CSV 文件

## 数据规模

- 去重字头：20,596 个
- 总记录（含异读）：26,468 条
- 上古音覆盖：66.1%
- 上古·开合等第覆盖：66.5%（以《汉字古音表稿》Word 表格标题为准，如「幼」伊謬=幽部開四[iəu]、（集）一笑-笑=幽部開三[ĭəu]）
- 表稿反切覆盖：66.0%（来自《汉字古音表稿》，与广韵反切互为补充）
- 韵部拟音覆盖：99.8%（以《汉字古音表稿》Word 表格标题中的上古韵部拟音为准）
- 表稿集韵异读：37 条（含释义括注，其中 11 条已拆分为独立记录分多行显示，异读的开合等第与拟音独立标注）
- 中古音覆盖：100.0%

## 使用方法

1. 在输入框中粘贴要查询的汉字
2. 点击「查询」按钮（或按 Ctrl+Enter）
3. 查看表格中的音韵地位信息
4. 点击「导出CSV」保存查询结果

## 数据来源

- 上古音：《汉字古音表稿》
- 中古音：《广韵》（周祖谟校本）
